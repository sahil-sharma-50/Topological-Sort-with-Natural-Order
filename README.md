# Topological Sort with Natural Order

This repository contains two solutions for the Topological Sorting with Natural Order preference coding problem.

1. **`topological_sort_bruteforce.py`**: A brute-force approach
2. **`topological_sort_dfs.py`**: An optimized solution using **DFS-based Topological Sort**
   - Reference:  
     [Topological Sort using DFS](https://www.geeksforgeeks.org/dsa/topological-sort-using-dfs/)
